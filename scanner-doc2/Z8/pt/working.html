Modo de trabalho#

Na área de trabalho fora do limite de 2. Receptor G ou sinal Bluetooth, você pode ativar o modo de armazenamento do scanner, seguindo as etapas descritas abaixo. Neste modo, todos os dados digitalizados serão armazenados diretamente no scanner. Antes de enviar os dados, será guardado permanentemente no scanner para que você possa carregar os dados quando o scanner estiver no intervalo de sinal de recebimento.

Modo normal#

Nota

Defina para o modo normal e os dados serão enviados imediatamente após a verificação.

Modo de armazenamento#

Nota

Defina o modo de armazenamento. Após uma verificação bem-sucedida, os dados não serão carregados imediatamente, mas serão armazenados no scanner.

Envio de dados[1]#

Nota

Faça o upload dos dados armazenados no scanner.

Quantidade total de armazenamento#

Nota

Verifique quantos pedaços de dados são armazenados.

Limpar o código de barras armazenado#

Nota

Limpe os dados armazenados no scanner.

Desativar modo de armazenamento automático#

Ativar Modo de Armazenamento Automático[2]#

[1]

Durante o carregamento de dados, escanear o comando «Upload de dados» novamente cancelará / interromperá o envio de dados.

[2]

Em modo de transmissão sem fio, quando o modo de armazenamento automático é ativado, o código de barras será automaticamente armazenado quando a transmissão falhar, e os dados armazenados podem ser obtidos através de «Carregamento de dados».
