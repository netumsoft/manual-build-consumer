Modo de escaneo#

Modo lanzador de teclas#

Pulse el botón para activar el escaneo, y suelte el botón para finalizar la exploración. El escaneo termina cuando el escaneo es exitoso o el tiempo de lectura excede el tiempo de lectura simple.

Modo continuo#

El escáner funciona continuamente. Si el escaneo tiene éxito o el tiempo de escaneo excede el tiempo de trabajo único, el escaneo actual termina. Si el tiempo excede el tiempo especificado, el siguiente escaneo se activa automáticamente.

Modo de autodetección#

En el modo de detección automática, el escáner detectará el brillo del entorno circundante. Cuando el brillo cambia, el escaneo se activa. El escaneo termina cuando el escaneo es exitoso o el tiempo de escaneo excede el tiempo de trabajo único. Independientemente de si el último escaneo fue exitoso o falló, se volverá a detectar el brillo del entorno circundante.
