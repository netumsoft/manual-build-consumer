Transmettre le préambule#

Les caractères préambules (code pays et caractères système) peuvent être transmis dans le cadre de la symbologie UPC-A. Sélectionnez l’une des options suivantes pour définir et transmettre un préambule UPC-A à l’hôte : Transmettre uniquement les caractères système Transmettre les caractères système et le code de pays (« 0 » pour les États-Unis), ou Transmettre aucun préambule.

Transmettre le préambule#

Les caractères préambules (code pays et caractères système) peuvent être transmis dans le cadre de la symbologie UPC-A. Sélectionnez l’une des options suivantes pour définir et transmettre un préambule UPC-A à l’hôte : Transmettre uniquement les caractères système Transmettre les caractères système et le code de pays (« 0 » pour les États-Unis), ou Transmettre aucun préambule.

Transmettre le caractère de vérification#

Convertir l’UPC-E en UPC-A#

Lorsque Enable est sélectionné, Convertit les données décodées en UPC-E au format UPC-A. Après la conversion, les données suivent le format UPC-A et sont affectées par les sélections de programmation UPC-A (par exemple, le préambule, vérifiez le chiffre).

Extension EAN-8#

Lorsque Enable est sélectionné, le scanner ajoute cinq zéros principaux au code-barres EAN-8 décodé pour étendre à 13 chiffres.

Note

  • Désactiver:Quand le scanner scanne un code-barres avec un code supplémentaire, il décode le code-barres et ignore le code additionnel. Le scanner peut également décoder des codes à barres sans le code additionnel.

  • Activer:Scanner ne décode que les codes à barres mixtes avec des codes additionnels.

  • Adaptive:Scanner décode un mélange de code-barres avec et sans code-module.

Code 32#

Convertir le code 39 en code 32 (code pharmaceutique italien)#

Le Code 32 est une variation du Code 39 utilisé par l’industrie pharmaceutique italienne. Scannez le code-barres approprié ci-dessous pour activer ou désactiver la conversion de Code 39 en Code 32.

Préfixe Code 32#

Activer ce paramètre ajoute le caractère de préfixe « A » à tous les codes Code 32. Avant d’activer ce paramètre, vous devez convertir le code 39 en code 32 (code pharmaceutique italien)

Code 11#

Activer/Désactiver#

Transmettre le caractère de vérification#

Vérifier la vérification des chiffres#

Cette fonction permet au scanner de vérifier l’intégrité de tous les symboles Code 11 pour vérifier que les données sont conformes à un algorithme de contrôle spécifié. Ceci sélectionne le mécanisme des chiffres de vérification pour le code à barres décodé Code 11 . Les options incluent la vérification d’un chiffre de vérification, la vérification de deux chiffres de vérification, ou la désactivation de la fonctionnalité.
