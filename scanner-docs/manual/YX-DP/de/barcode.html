UPC-A#

Aktiviere/Deaktivieren#

Präambelzeichen übertragen#

Präambelzeichen (Landesvorwahl und Systemzeichen) können im Rahmen der UPC-A-Symbologie übertragen werden. Wählen Sie eine der folgenden Optionen, um eine UPC-A-Präambel zu setzen und zu übertragen: Nur Systemzeichen übertragen, Übertragen Sie Systemzeichen und Ländercode („0“ für die Vereinigten Staaten) oder übertragen Sie keine Präambel.

Präambelzeichen übertragen#

Präambelzeichen (Landesvorwahl und Systemzeichen) können im Rahmen der UPC-A-Symbologie übertragen werden. Wählen Sie eine der folgenden Optionen, um eine UPC-A-Präambel zu setzen und zu übertragen: Nur Systemzeichen übertragen, Übertragen Sie Systemzeichen und Ländercode („0“ für die Vereinigten Staaten) oder übertragen Sie keine Präambel.

Transmit-Prüfzeichen#

UPC-E in UPC-A umwandeln#

Wenn Enable ausgewählt ist, konvertiert UPC-E dekodierte Daten in das UPC-A-Format. Nach der Konvertierung folgen die Daten dem UPC-A-Format und sind von der UPC-A-Programmierauswahl (z.B. Präambel, Prüfziffer) betroffen.

Wenn Enable ausgewählt ist, fügt der Scanner dem dekodierten EAN-8 Barcode fünf führende Nullen hinzu, um sich auf 13 Ziffern zu erweitern.

UPC/EAN/JAN-Add-on-Code#

Bemerkung

  • Deaktivieren:Wenn der Scanner einen Barcode mit einem zusätzlichen Code durchsucht, entschlüsselt er den Barcode und ignoriert den zusätzlichen Code. Der Scanner kann auch ohne zusätzlichen Code Barcodes dekodieren.

  • Aktivieren:Scanner dekodiert nur gemischte Barcodes mit zusätzlichen Codes.

  • Adaptive:Scanner dekodiert eine Mischung aus Barcodes mit und ohne Add-on-Codes.

Code 39 zu Code 32 konvertieren (italienischer pharmazeutischer Code)#

Code 32 ist eine von der italienischen pharmazeutischen Industrie verwendete Variante des Kodex 39. Scannen Sie den entsprechenden Barcode unten, um die Umwandlung von Code 39 in Code 32 zu aktivieren oder zu deaktivieren.

Code 32 Präfix#

Durch das Aktivieren dieses Parameters wird das Präfix „A“ zu allen Code 32 Codes hinzugefügt. Bevor Sie diesen Parameter aktivieren, müssen Sie Code 39 in Code 32 (italienischer Pharmakodex) konvertieren

Aktiviere/Deaktivieren#

Transmit-Prüfzeichen#

Überprüfe Ziffernprüfung#

Diese Funktion erlaubt es dem Scanner die Integrität aller Code 11 Symbologien zu überprüfen, um zu überprüfen, ob die Daten mit einem bestimmten Prüfziffer-Algorithmus übereinstimmen. Damit wird der Prüfziffer-Mechanismus für den dekodierten Code 11 Barcode ausgewählt. Dazu gehören das Überprüfen einer Prüfziffer, das Überprüfen zweier Prüfziffern oder das Deaktivieren der Funktion.
