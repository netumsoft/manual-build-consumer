Paramètres du personnage de départ#

Le caractère de début est ajouté avant les données décodées, et le format est : le caractère de début + les données décodées.

Paramètres de caractère de fin#

Le caractère de fin est ajouté après le décodage des données au format des données décodées + caractère de fin.
