UPC-A#

Ativar/Desativar#

Transmitir Caractere Preâmbulo#

Caracteres preâmbulos (código do país e caracteres do sistema) podem ser transmitidos como parte da simbologia UPC-A. Selecione uma das seguintes opções para definir e transmitir um preâmbulo UPC-A para o host: somente caracteres do sistema de transmissão, Transmitir caracteres do sistema e código do país («0» para os Estados Unidos), ou não Transmitir nenhum preâmbulo.

Caracteres preâmbulos (código do país e caracteres do sistema) podem ser transmitidos como parte da simbologia UPC-A. Transmitir verificação de personagem#

Converter UPC-E para UPC-A#

Quando Ativar está selecionada, converte dados de UPC-E decodificados para o formato UPC-A. Após a conversão, os dados seguem o formato UPC-A e são afetados por seleções de programação UPC-A (por exemplo, preâmbulo, verificação digital).

Extensão EAN-8#

Quando Enable é selecionado, o scanner adiciona cinco zeros à direita no código de barras EAN-8 decodificado para se estender a 13 dígitos.

Nota

  • Desabilitar: Quando o scanner escanea um código de barras com um código adicional, ele decodifica o código de barras e ignora o código adicional. O scanner também pode decodificar códigos de barras sem o código adicional.

  • Ativar: O Leitor QR decodifica apenas códigos de barras mistos com códigos adicionais.

  • Adaptivo: Scanner decodifica uma mistura de códigos de barras com e sem códigos de complementos.

Código 32#

Converter o Código 39 para o Código 32 (Código Farmacêutico Italiano)#

O Código 32 é uma variação do Código 39 utilizado pela indústria farmacêutica italiana. Escaneie o código de barra abaixo para ativar ou desativar a conversão de Código 39 para Código 32.

Prefixo do código 32#

Habilitando este parâmetro adiciona o caractere de prefixo «A» para todos os códigos de código 32. Antes de ativar este parâmetro, você deve converter o Código 39 para 32 (Código Farmacêutico Italiano)

Código 11#

Ativar/Desativar#

Transmitir verificação de personagem#

Verificar Verificação de Dígitos#

Este recurso permite que o scanner verifique a integridade de todos os símbolos do Código 11 para verificar se os dados estão em conformidade com um algoritmo de dígito de verificação especificado. Isto seleciona o mecanismo de verificação de dígitos para o código de barras de código 11 decodificado. Opções incluem a verificação de um dígito de verificação, checando dois dígitos ou desativando o recurso.
